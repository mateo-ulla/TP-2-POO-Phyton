# Gestión de Herramientas en Python 

Este proyecto en Python permite registrar información básica sobre herramientas utilizando una clase `Nodo`. Al ejecutarlo, el usuario puede ingresar los datos de tres herramientas y luego ver un resumen con todos los detalles registrados.

## Características

- Ingreso interactivo de:
  - Nombre de la herramienta
  - Marca
  - Precio
  - Cantidad
- Visualización formateada de todas las herramientas ingresadas

## Estructura del Código

El programa se basa en la clase `Nodo`, que contiene:

- `__init__`: Método constructor que recibe un nombre de nodo.
- `mostrar_datos`: Método que solicita al usuario los datos de las herramientas, los almacena en una lista y luego los imprime de forma ordenada.
